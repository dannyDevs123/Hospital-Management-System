 MediCare Hospital Management System

A robust, modern desktop application designed to streamline healthcare workflows, handle session configurations, and manage clinic operational data. 

 Features
* **Splash Screen UI:** A clean, engaging user interface presented on application startup.
* **Session Management:** Secure tracking of active user states across administrative and clinical modules.
* **Comprehensive Dashboards:** Structured views for managing appointments, billing, doctors, patients, and reporting.

 Tech Stack
* **Language:** Java
* **UI Framework:** JavaFX w
* **Architecture:** MVC (Model-View-Controller) pattern

 📁 Project Structure
* `com.hospital.controller` - Manages UI logic (Appointments, Billing, Doctors, Splash Screen, etc.)
* `com.hospital.dao` - Data Access Object layer for handling database transactions
* `com.hospital.model` - Core data models and entities
* `com.hospital.service` - Business logic and core backend services (e.g., `SessionManager`)
* `com.hospital.util` - Utility classes and helper functions
